Wbrew pozorom - jeżeli chcemy zastosować opóźnienie, od tego trzeba zacząć. Zmiany, które wprowadzimy mogą mieć wpływ na integrację.

Ponieważ właściwość `display` nie może być animowana, trzeba docelowy `display:flex;` przenieść do właściwości CSS elementu div przed najechaniem i manipulować elementem przy pomocy zmieniającej się właściwości `opacity`. Do tego trzeba dodać `left:-100%;` żeby uniknąć efektu stopniowego znikania. Natomiast sama zmiana opacity będzie spowolniona przez `transition: opacity .5s ease-in-out;`,

### Mega Menu + Hamburger

Na samym początku trzeba się zastanowić jak ma wyglądać mechanika menu w formie Hamburgera. Na ten moment całą treść bloku Mega Menu mamy ukrytą. Można utworzyć osobną podstronę i tam przekierowywać użytkownika, ale jest to po pierwsze mało wydajne, a po drugie mylące dla użytkownika.

Rozwiązaniem jest submenu (menu drugiego rzędu).

Najprościej zrobić to tak jak jest w menu [Adobe](https://www.adobe.com/), po kliknięciu submenu rozwija się przesuwając resztę menu w dół i zamykając ewentualnie poprzednio otwarte sumbenu. Jest to harmonijka. To jest dobre rozwiązanie jeśli submenu nie ma wielu elementów.

Bardziej ambitne i zaawansowane jest takie, które nasuwa się na rozwiniętego Hamburgera i tak samo się zachowuje, żeby je schować trzeba kliknąć iksa: jest to Hamburger w Hamburgerze. Przykładem może być menu zastosowane na stronie [Atlassian](https://www.atlassian.com/).

#### Hamburger + Dropdown

Zacznijmy od pierwszego rozwiązania. Najprościej jest zrobić to samym CSS-em i takim przykładowym kodem jest zmiana poprzedniej wersji w tym fragmencie:

```css {numberLines}
@media (max-width: 700px) {
  .menu-dropdown ul li div {display:none; opacity:0; transition: opacity .5s ease-in-out;}
  .menu-dropdown ul li:hover div {display: flex; flex-direction: column; position:relative; opacity:1;}
}
```

Ma to dwie wady:

- zmiana następuje błyskawicznie i może to być mylące
- nie ma oczywistego wyjścia z wyświetlenia submenu

Trzeba to zrobić na kliknięcie. Pierwsze powoduje wysunięcie podmenu, drugie jego zamknięcie. W tym celu umieszczamy w odnośniku menu nadrzędnego wywołanie funkcji która dodaje i usuwa klasę CSS z następnego elementu HTML:

```js {numberLines}
//
let flag = true;
const Menu = () => {

    const handleSubmenu = (e) => {
        const divSubmenu = e.target.nextElementSibling;
        if (flag) { divSubmenu.classList.add("expanded"); flag = false; console.log(divSubmenu.classList) }
        else { divSubmenu.classList.remove("expanded"); flag = true; }
    }

    return (<nav className="menu-dropdown">
        <ul>
            <li><Link to="/">Home</Link></li>
            <li><Link to="#" onClick={(e) => handleSubmenu(e)}>Articles</Link>
                <div>
                  //
```
I analogiczna zmiana tego samego fragmentu kodu CSS:

```css {numberLines}
@media (max-width: 700px) {
  .menu-dropdown ul li > div {display:none; }
  .menu-dropdown ul li > div.expanded {display: flex; flex-direction: column; position:relative; left:0; opacity:1;}
}

@media (min-width: 700px) {
    .menu-dropdown > ul {flex-direction: row;}
    .menu-dropdown ul li:hover > div { width:100%; max-width:99vw;}
}
```
